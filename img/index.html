Efeito Monalisa: arte interativa com P5.js>

Visa criar uma versão interativa da famosa pintura, onde o rosto da Mona Lisa parece seguir o movimento do mouse. O projeto utiliza a biblioteca P5.js para manipular a imagem e adicionar interatividade. A ideia é que, ao mover o mouse pela tela, os olhos e a direção do olhar da Mona Lisa se ajustem dinamicamente, criando a ilusão de que ela está sempre olhando para o usuário.

Quente e Frio: interação no P5.js

É um exercício interativo onde o usuário move um círculo na tela e recebe feedback sobre a distância em relação a um ponto secreto, sendo "quente" (próximo) ou "frio" (distante). O projeto utiliza conceitos como coordenadas aleatórias, variáveis para posicionamento e interação do usuário através do mouse ou teclado. É uma excelente forma de aprender os conceitos básicos de programação interativa, como eventos, coordenadas, cálculo de distância e feedback visual. É um projeto relativamente simples, mas que permite explorar diversas possibilidades e personalizar a experiência do jogo.

Jogo de Corrida: Listas, funções e repetição

Essa programação no Alura tem duas partes e usa o p5.js para ensina a criar um jogo de corrida do zero, utilizando a biblioteca JavaScript p5.js. As aulas abordam os conceitos fundamentais da programação de jogos, como movimentação de objetos, detecção de colisão e interface do usuário, adaptando esses conceitos para o desenvolvimento de um jogo de corrida.
